my emacs configuration

./install.sh

# shortcut

[f1] menu bar
[f2] create ctags
[f3] search string
[f4] search tag
[f5] reverse f4
...
[f10] show line number at the left side
[f11] show 80 column line
[f12] highlight char after column 80
